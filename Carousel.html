The Objective

Turn the carousel into two rows that move simultaneously with the navigation buttons. Images should not repeat and images added to the UGC gallery that are drawn by the API should add to the carousel. ie. The content should remain dynamic and not be made into static content. Full responsive compatibility should be maintained.

Here is a video showing how the slider should operate when clicking the nav buttons and what happens when you reach either end. https://www.youtube.com/watch?v=AIfGHefVQ0Y

This can be achieved a number of ways. If you are not successful explain your logic as to how you got to where you did.
